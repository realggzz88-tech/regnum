# Assets Directory

This folder contains visual assets for the Regnum Consulting website.

## Required Assets

### 1. favicon.ico
- **Purpose**: Browser tab icon
- **Format**: .ico file
- **Recommended size**: 32x32px or 16x16px
- **Location**: `assets/favicon.ico`
- **How to create**: 
  - Design a simple icon representing your brand
  - Convert to .ico format using online tools like favicon.io or realfavicongenerator.net

### 2. og-image.jpg
- **Purpose**: Social media preview image (Open Graph & Twitter Card)
- **Format**: .jpg or .png
- **Recommended size**: 1200x630px
- **Location**: `assets/og-image.jpg`
- **Design guidelines**:
  - Include company name "REGNUM CONSULTING"
  - Use brand colors (#0B0F14 background, #C7E35F accent)
  - Keep text readable and centered
  - Professional, minimal design
  - This image appears when your website is shared on social media

### 3. logo.png (Optional)
- **Purpose**: Company logo for navbar (currently using text)
- **Format**: .png with transparent background
- **Location**: `assets/logo.png`
- **If you want to use an image logo instead of text**:
  - Update the navbar in index.html to use `<img src="assets/logo.png" alt="Regnum Consulting">`
  - Recommended height: 40-50px
