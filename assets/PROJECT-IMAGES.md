# Project Placeholder Image

This folder should contain project screenshots and media.

## Required for Template

- `project-placeholder.jpg` - Default placeholder image (1200x800px)
  - Currently missing - template will show broken image until real project images are added

## For Each Project

Create project-specific images:
- `project-[name]-1.jpg` - Main screenshot
- `project-[name]-2.jpg` - Additional view (optional)
- `project-[name]-dashboard.jpg` - If applicable

## Recommended Specs

- **Format**: JPG or PNG
- **Dimensions**: 1200x800px minimum (maintains 3:2 aspect ratio)
- **File Size**: < 500KB (optimize for web)
- **Quality**: 80-85% compression

## Tools for Image Optimization

- TinyPNG (https://tinypng.com/)
- Squoosh (https://squoosh.app/)
- ImageOptim (Mac)
